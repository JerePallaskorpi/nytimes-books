# nytimes-books

## Install Instructions

- Clone the project
- Client
  - $ cd ../client
  - $ npm i
  - $ npm start
  - Client should be running at http://localhost:3000
- Server
  - $ cd ../server
  - $ cp .env.example .env
  - Add API key to .env file
    - https://developer.nytimes.com/get-started
  - $ npm i
  - $ npm start
  - Server should be running at http://localhost:4000
